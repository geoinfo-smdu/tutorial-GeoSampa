GeoSampa tutorial internet

Bem-vindo ao GeoSampa!

Este tutorial auxilia no uso das ferramentas disponíveis e na navegação pelos mapas e camadas de dados do GeoSampa.

http://geosampa.prefeitura.sp.gov.br/

Há três formas de usar e visualizar os dados do GeoSampa:

  1. Usando o site diretamente, seguindo o link dado acima
  2. Usando programas:
    1. Offline, baixando os arquivos
    2. Online, usando serviços web para acessar as camadas (WMS e WFS)
